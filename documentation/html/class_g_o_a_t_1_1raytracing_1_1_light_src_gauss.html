Detailed Description

This class describes a focused gaussian beam.

Class which describes a (focused) gaussian light source. The main direction is given by the source position and the focal position. The electric field is calculated by The waist of the beam is only used for the correct electric field distribution within the starting area. Since we are working with geometrical optics the rays follow straight lines inside the medium.
