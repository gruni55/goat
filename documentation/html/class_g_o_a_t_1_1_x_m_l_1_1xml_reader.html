Detailed Description

This class provides functionality to read a XML-file This function reads the XML file . It can also perform calculations. Beware: The parser is case sensitive. Element names starting with capital letters (except of "real" and "imag" for complex numbers), whereas values are always lower case The structure of the XML-File is as follows (the order of the subdivision may be changed): Signs: o: optional, n: necessary, i: independend parameters, i.e. this parameter is valid for all types Values: float: floating point number int: integer number bool: possible values: true or false.

Structure of a XML-File
Root Value
Scene
Objects
Object
ni Type "ellipsoid","surface","cone",
"aspheric_lens" or "spheric_lens"
oi Alpha float
oi Beta float
oi Gamma float
oi IsActive bool
oi n
oin* real double
oin* imag double
oi Position
oin* x double
oin* y double
oin* z double
