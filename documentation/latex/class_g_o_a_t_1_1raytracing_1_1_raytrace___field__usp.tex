\doxysection{GOAT\+::raytracing\+::Raytrace\+\_\+\+Field\+\_\+usp Class Reference}
\hypertarget{class_g_o_a_t_1_1raytracing_1_1_raytrace___field__usp}{}\label{class_g_o_a_t_1_1raytracing_1_1_raytrace___field__usp}\index{GOAT::raytracing::Raytrace\_Field\_usp@{GOAT::raytracing::Raytrace\_Field\_usp}}


This class provides a raytracer to calculate the field distribution in a box detector for a ultrashort pulsed light source Unlike the class \doxylink{class_g_o_a_t_1_1raytracing_1_1_raytrace__usp}{Raytrace\+\_\+usp}, the field in more than one object or in parts of objects can stored. As in \doxylink{class_g_o_a_t_1_1raytracing_1_1_raytrace__usp}{Raytrace\+\_\+usp}, in the \doxylink{class_g_o_a_t_1_1raytracing_1_1_super_array}{Super\+Array} grid the information (step length, index of the object involved in the step) for each step from the light source to the corresponding point in the grid is stored. For each reflection order one grid will be created. The field will be stored within one or more boxes (called "{}box detector"{})

\doxysubsection{Detailed Description}
This class provides a raytracer to calculate the field distribution in a box detector for a ultrashort pulsed light source Unlike the class \doxylink{class_g_o_a_t_1_1raytracing_1_1_raytrace__usp}{Raytrace\+\_\+usp}, the field in more than one object or in parts of objects can stored. As in \doxylink{class_g_o_a_t_1_1raytracing_1_1_raytrace__usp}{Raytrace\+\_\+usp}, in the \doxylink{class_g_o_a_t_1_1raytracing_1_1_super_array}{Super\+Array} grid the information (step length, index of the object involved in the step) for each step from the light source to the corresponding point in the grid is stored. For each reflection order one grid will be created. The field will be stored within one or more boxes (called "{}box detector"{})
