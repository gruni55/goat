\doxysubsection{Detailed Description}
This class describes a focused gaussian beam. 

Class which describes a (focused) gaussian light source. The main direction is given by the source position and the focal position. The electric field is calculated by $\vec E(r,z)=\vec E_0 \frac{w_0}{w(z)}\cdot e^{\frac{r^2}{w^2(z)}}\cdot e^{-ik\frac{r^2}{2R(z)}}\cdot e^{i(\zeta(z)-kz)}$ The waist of the beam is only used for the correct electric field distribution within the starting area. Since we are working with geometrical optics the rays follow straight lines inside the medium.
