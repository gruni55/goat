\doxysubsection{Detailed Description}
An attribute is a name-\/value pair. Elements have an arbitrary number of attributes, each with a unique name.

\begin{DoxyNote}{Note}
The attributes are not XMLNodes. You may only query the \doxylink{classtinyxml2_1_1_x_m_l_attribute_aa8c7cf4c94a636ae75046658e551614e}{Next()} attribute in a list. 
\end{DoxyNote}
