Accept a hierarchical visit of the nodes in the Tiny\+XML-\/2 DOM. Every node in the XML tree will be conditionally visited and the host will be called back via the \mbox{\hyperlink{classtinyxml2_1_1_x_m_l_visitor}{XMLVisitor}} interface.

This is essentially a SAX interface for Tiny\+XML-\/2. (Note however it doesn\textquotesingle{}t re-\/parse the XML for the callbacks, so the performance of Tiny\+XML-\/2 is unchanged by using this interface versus any other.)

The interface has been based on ideas from\+:


\begin{DoxyItemize}
\item \href{http://www.saxproject.org/}{\texttt{ http\+://www.\+saxproject.\+org/}}
\item \href{http://c2.com/cgi/wiki?HierarchicalVisitorPattern}{\texttt{ http\+://c2.\+com/cgi/wiki?\+Hierarchical\+Visitor\+Pattern}}
\end{DoxyItemize}

Which are both good references for \char`\"{}visiting\char`\"{}.
