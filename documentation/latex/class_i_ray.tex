\doxysubsection{Detailed Description}
This class represents a single ray. 

This ray is intended to calculate with a single ray with two different polarisations. This class should not be used directly. It is intended for use together with classes derived by \mbox{\hyperlink{class_light_src}{Light\+Src}} (like \mbox{\hyperlink{class_light_src_plane}{Light\+Src\+Plane}} or \mbox{\hyperlink{class_light_src_gauss}{Light\+Src\+Gauss}})
