\doxysubsection{Detailed Description}
Structure, which acts as a container for all informations needed to process the calculation. ~\newline
 

This structure is needed for the Fourier transformation which is used to calculate the temporal development of the electric field in space. Since dispersion is considered, for all objects and also for the surrounding medium functions are necessary which calculate the complex-\/valued refractive index from the wavelength. These functions are stored in the element n\+List.
