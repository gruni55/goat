\doxysubsection{Detailed Description}
Class defining a scene with lightsources and objects. This is a container used to inform the Raytracer about all necessary settings. Here, all informations about light sources and objects are stored. Light sources are described by classes derived from the virtual \mbox{\hyperlink{class_light_src}{Light\+Src}} base class. All objects are described by classes derived from the virtual class \mbox{\hyperlink{class_object_shape}{Object\+Shape}}.
