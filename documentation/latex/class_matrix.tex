\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Matrix$<$ T $>$}

This class represents a threedimensional (numeric) \mbox{\hyperlink{class_matrix}{Matrix}} as Template. 

The \mbox{\hyperlink{class_matrix}{Matrix}} class represents a template class for 3x3 numeric matrices. Operators for standard matrix operation are provided. This class is intended to work with the \mbox{\hyperlink{class_vector}{Vector}} template class defined in this S\+DK.
