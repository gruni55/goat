\doxysubsection{Detailed Description}
This class calculates the time dependence of a field which calculated before. 

The calculation of the time dependence is made in two steps. In the first step, the fields are calculated for different wavelengths. In this class, the second step will be made. Here, the time dependence is calculated with a kind of Fourier transformation. The class requires an array of \doxylink{class_g_o_a_t_1_1raytracing_1_1_super_array}{Super\+Array} elements which contain the the information over all steps for every ray (length of the step and material). The process assumes, that the wavelength range can be subdivided into subranges in which the paths of the rays remain nearly the same. Since the phase is much more sensitive to changes of the refractive index, the phase change within these subranges is considered. The structure \doxylink{struct_g_o_a_t_1_1raytracing_1_1_trafo_parms}{Trafo\+Parms} carries all information needed for the calculation. The spectral resolution, needed to prevent \"{}ghost\"{} peaks is a function of the time difference dt to a certain reference time tref\+: dt=\texorpdfstring{$\vert$}{|}tref-\/t\texorpdfstring{$\vert$}{|}. By default tref is set to 0, but if you are interested of a range around a certain time, set tref to this time to reduce the amount of needed frequencies
