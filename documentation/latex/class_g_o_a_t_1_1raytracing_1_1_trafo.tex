This class calculates the time dependence of a field which calculated before The calculation of the time dependence is made in two steps. In the first step, the fields are calculate for different wavelengths. In this class, the second step will be made. Here the time dependence is calculated with a kind of Fourier transformation. The class requires an array of \mbox{\hyperlink{class_g_o_a_t_1_1raytracing_1_1_super_array}{Super\+Array}} elements which contain the fields for the different wavelengths. The process assumes, that the wavelength range can be subdivided into subranges where the paths of the rays remain nearly the same. Since the phase is much more sensitive to changes of the refractive index, only the phase change within this subranges is considered. The structure \mbox{\hyperlink{struct_g_o_a_t_1_1raytracing_1_1_trafo_parms}{Trafo\+Parms}} carries all information needed for the calculation.
